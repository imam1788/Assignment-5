Question-1: What are the different ways to select an element in the DOM?

There are different ways to select an element in the DOM. Example: getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), querySelectorAll().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

'innerHTML' gets HTML content including tags, 'innerText' gets only visible text and 'textContent' returns all text including hidden elements.

Question-3: What is event delegation in the DOM?

Event delegation is when a parent element listens for events on its children, so instead of setting up separate listeners for each child, the parent catches events as they bubble up from the kids.

Question-4: What is event bubbling in the DOM?

When an event happens in the DOM, it doesn't just stay at the element where it occurred; it bubbles up to its parent elements, allowing those parents to react to the event too..

Question-5: How do you create, add, and remove elements using JavaScript?

To work with elements in JavaScript, you can create them with createElement(), add them using appendChild(), and remove them with removeChild().
